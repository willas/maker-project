# 智能桌面清洁小车 — 视频介绍文案

> 总时长控制在 **2分30秒** 左右（留余量），语速自然别赶
>
> 两个小朋友轮流说，标注 **A** 和 **B**
>
> 【】里面是画面/动作提示，不用念

---

## 片头（10秒）

【画面：项目名称字幕卡，手写或打印都行】

```
项目：智能桌面清洁小车
组别：______（填你们的组别）
学校：______
学生：______ 、 ______
```

---

## 第一段：开场介绍（30秒）

【画面：两个小朋友站在桌子旁边，小车放在桌上】

**A：** 大家好！我是______，

**B：** 我是______。

**A：** 每天写完作业，桌子上都是橡皮屑、铅笔灰，特别烦。

**B：** 我也是！每次都要用手扒拉半天，还扒拉不干净。

**A：** 所以我们就想——能不能做一个小车，帮我们自动打扫桌面？

**B：**【指向小车】就是它！我们的智能桌面清洁小车！

---

## 第二段：功能介绍（40秒）

【画面：拿起小车，展示各个部分】

**A：** 它有这些功能——

**A：**【指着底部的刷子和风扇】第一，底下有一个旋转毛刷和一个小鼓风机。刷子把碎屑扫起来，鼓风机把碎屑吸进收集盒里。

**B：**【指着前面的传感器】第二，前面有超声波传感器，碰到杯子、书本会自动绕开。

**A：**【把小车翻过来，指着底部三个传感器】第三，底下有三个悬崖传感器，到了桌子边缘会自动后退转弯，不会掉下去。

**B：** 第四，它还能听懂我们说话！我说"打扫"它就开始扫，说"停下"它就停。

**A：** 最好玩的是——它会用**我们自己的声音**说话！

**B：** 对！里面的语音都是我们自己录的，还可以一键切换谁的声音。

---

## 第三段：现场演示（50秒）

【画面：桌面上撒一些橡皮屑】

**B：** 我们来演示一下。先在桌上撒一些橡皮屑。

【画面：碎屑撒好了，桌面看得出来是脏的】

**A：**【对着小车说】打扫！

【画面：小车开始跑，语音播报"出发咯！让我把桌子擦得亮亮的！"】

**B：** 它开始扫了！你看，刷子在转，风扇在吸。

【画面：小车跑到桌边，后退转弯】

**A：** 看！到桌子边了，它自己后退了，还说"哎呀好险好险"，哈哈。

【画面：小车碰到杯子/书本，绕开】

**B：** 碰到杯子也会绕开。

【等小车扫几秒】

**A：**【对着小车说】停下！

【画面：小车停了，语音播报"好的好的，我休息一下~"】

**B：**【拿起小车，拉出收集盒展示】你看，碎屑都在收集盒里面！

【画面：特写收集盒里面的碎屑，再看看干净的桌面】

**A：** 桌面干干净净了！

---

## 第四段：我们做了什么（20秒）

【画面：展示制作过程的照片，或者对着镜头说】

**B：** 这个小车是我们自己组装的。我负责接线、装轮子、装传感器。

**A：** 我录了所有的语音台词，还设计了外壳。编程的部分我们跟着爸爸/老师一起学的。

**B：** 毛刷是我们用瓶盖和扎带自己做的！

---

## 第五段：结尾（10秒）

【画面：两个小朋友和小车合影】

**A：** 以后写完作业——

**B：** 按一下就干净！

**合：** 谢谢大家！

【小车播放语音："你好呀！我是桌面小管家~"】

## 拍摄提示

| 项目 | 建议 |
|------|------|
| 拍摄设备 | 手机横屏拍，720p 就行 |
| 光线 | 白天靠窗拍，光线充足 |
| 背景 | 干净的桌面，不要太杂乱 |
| 声音 | 安静的房间，关窗关电视 |
| 字幕 | 可以不加，加了更好 |
| 时长 | 练几遍控制在 2分30秒内 |
| 剪辑 | 手机自带剪辑就行，不需要花哨特效 |
| 文件 | 导出 720p MP4，改名为 **视频.mp4** |

## 练习建议

1. 先各自把台词念熟（不用背死，意思到了就行，自然最重要）
2. 对着手机练 2-3 遍，看看哪里卡壳
3. 演示环节提前跑一遍，确保小车正常工作
4. 正式拍的时候，忘词了没关系，剪掉重来那一段就行
