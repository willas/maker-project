# 万向轮（万向球轮）

## 是什么

一个**金属小球装在底座里**，可以朝任意方向滚动。装在小车尾部底部做被动支撑轮，配合前面两个 N20 驱动轮，形成稳定的三角形着地。

## 长什么样

```
直径 10–15mm，一颗弹珠大小

    ┌───────┐ ← 金属底座（有螺丝孔，固定到车底）
    │  (●)  │ ← 金属球（可以 360° 任意方向滚动）
    └───────┘
```

## 在清洁小车里的位置

```
         前进方向 ↑

    左驱动轮 ○         ○ 右驱动轮
              ╲       ╱
               ╲     ╱
                ╲   ╱
                 (●)        ← 万向轮在车尾
```

三个着地点形成**三角形**——最稳定、最简单的底盘结构。

## 为什么不用第三个电机轮

| 方案 | 优点 | 缺点 |
|------|------|------|
| 2 驱动轮 + 1 万向轮 | **简单**、省钱、转向灵活 | 万向轮摩擦力小 |
| 3 个驱动轮 | 力气大 | 复杂、贵、需要 3 路驱动 |
| 4 个驱动轮 | 最稳 | 太复杂，桌面清洁不需要 |

扫地机器人、送餐机器人大多数都是 2 驱动轮 + 1 万向轮的布局，经典可靠。

## 万向轮不需要接线

它是纯机械的——没有电机、没有传感器、不接 ESP32，用螺丝固定到车底就行。

## 安装方式

```
车底板（亚克力 / 3D 打印）
    │
    │  ← M3 螺丝穿过底板
    │
  ┌─┴─┐
  │(●) │ ← 万向轮
  └───┘
```

用 M3 螺丝 + 螺母把万向轮底座固定到车尾底板即可。注意安装后万向轮底面要和两个驱动轮底面**在同一水平面**（可以用铜柱垫高调节）。

## 选购

淘宝搜 `万向球轮 金属 小 10mm` 或 `万向轮 小车 金属`，1–2 元。
- 选**金属球**的（塑料球容易卡住）
- 直径 10–15mm（太大会撑起车身、驱动轮悬空）
- 有的套装里自带万向轮，不需要单独买

## 一句话总结

**万向轮就是车尾的一个小金属球支撑轮**，能朝任何方向滚，不需要电、不需要接线，用螺丝固定到车底就行。配合两个驱动轮形成三角形底盘，1–2 元。
