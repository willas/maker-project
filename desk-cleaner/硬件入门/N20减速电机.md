# N20 减速电机

## 是什么

一个**拇指大小的小马达**，自带齿轮减速箱，转速慢但力气大，专门用来驱动小车轮子。

## 长什么样

```
大约 2.4cm × 1.2cm × 1cm，比一节 5 号电池还小

    ┌──────────────┐
    │  ┌────┐      │
    │  │齿轮│ 电机  │──── 输出轴（D型，接轮子）
    │  │减速│ 本体  │
    │  └────┘      │
    └──┤+  -├──────┘
       两根线（红+、黑-）
```

"N20"是这类小电机的型号代号，"减速"是因为里面有一组**微型齿轮箱**，把电机的高转速降下来、扭力放大。

## 为什么要"减速"

| | 不减速（直接用电机） | N20 减速电机 |
|--|---|---|
| 转速 | ~15000 RPM（疯转） | **100–200 RPM**（慢而稳） |
| 扭力 | 很小（带不动轮子） | **大**（能推动小车） |
| 效果 | 轮子空转打滑 | 轮子稳稳地走 |

就像自行车变速一样——低档骑得慢但踩得动上坡，高档踩得快但上不了坡。减速电机就是出厂自带"低档"。

## 关键参数怎么选

| 参数 | 桌面清洁小车推荐值 | 说明 |
|------|-------------------|------|
| **电压** | **6V** | 配 7.4V 电池经 TB6612 驱动刚好 |
| **转速** | **100–200 RPM** | 桌面小车不需要快，太快容易冲出桌边 |
| **轴型** | **D 型轴**（一面削平） | 方便套轮子不打滑 |
| **带轮子** | **选套装** | 省得自己配轮子，尺寸匹配 |

## 怎么接线

N20 电机只有 2 根线（红 + 黑 -），不是直接接 ESP32，而是接 **TB6612 的输出端**：

```
ESP32                TB6612               N20 电机
GPIO 25 ──→ AIN1 ┐                    ┌── 红线(+)
GPIO 26 ──→ AIN2 ├→ AO1 ─────────────┤   左电机
GPIO 32 ──→ PWMA ┘  AO2 ─────────────└── 黑线(-)

GPIO 27 ──→ BIN1 ┐                    ┌── 红线(+)
GPIO 14 ──→ BIN2 ├→ BO1 ─────────────┤   右电机
GPIO 33 ──→ PWMB ┘  BO2 ─────────────└── 黑线(-)
```

交换红黑线（或代码里改 IN1/IN2）就能反转。

## 正反转原理

TB6612 送出来的电流方向决定电机转向：

| 动作 | AO1 → AO2 电流方向 | 电机 |
|------|-------------------|------|
| 正转 | AO1 → 电机 → AO2 | 轮子向前 |
| 反转 | AO2 → 电机 → AO1 | 轮子向后 |
| 停止 | 无电流 | 不动 |

你不用管电流方向，代码里写 `HIGH/LOW` 就行，TB6612 自动处理。

## 买"带轮子套装"里有什么

淘宝搜 `N20 减速电机 6V 带轮子 套装`，一套约 6–10 元，一般包含：

```
┌─────────────────────────────┐
│  N20 电机 × 1               │
│  橡胶轮子 × 1（直径~34mm）   │
│  电机固定支架 × 1            │
│  螺丝 × 2                   │
└─────────────────────────────┘
```

清洁小车要买 **2 套**（左轮 + 右轮），总共约 12–20 元。

## 选购要点

- **电压选 6V**（不要 3V 的，力气不够；也不要 12V 的，太大了）
- **转速选 100–200 RPM**（太快小车在桌上不好控制）
- **选带轮子和支架的套装**（省事，尺寸匹配）
- **D 型轴**（轮子套上去不会空转）

## 一句话总结

**N20 减速电机就是一个自带"变速箱"的拇指小马达**，转得慢但有力气，配上橡胶轮子就能驱动小车。2 根线接 TB6612，代码控制正反转和速度，一套带轮子不到 10 块钱。
