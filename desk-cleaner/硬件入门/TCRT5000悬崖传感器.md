# TCRT5000 悬崖传感器（红外反射传感器）

## 是什么

一个**朝下安装的"眼睛"**，用红外线检测下方有没有桌面。有桌面 → 红外线反射回来 → 安全；到桌边了 → 红外线射进空气不回来 → 赶紧后退！

## 长什么样

```
大约 1.5cm × 1cm，一颗糖果大小

    ┌──────────┐
    │ TCRT5000 │
    │  ◐  ◑   │ ← 左边发射红外线，右边接收反射
    └──────────┘
     VCC DO GND  ← 3 根线（有的模块 4 根，多一个 AO 模拟输出）
```

## 工作原理

```
有桌面的时候：                    到桌边了：
    传感器                          传感器
    ◐→→→↓                         ◐→→→→→→  （射向空气，不回来）
    ◑←←←↑  反射回来                ◑  啥也没收到
   ━━━━━━━━━ 桌面                         ┃ 桌边
                                          ┃
   DO 输出 LOW（有桌面）              DO 输出 HIGH（悬空！）
```

## 在清洁小车里怎么用

装 **3 个**，朝下安装在车头底部前缘，呈扇形排列：

```
             前进方向
               ↑
    ◎─────────◎─────────◎
    左        中         右
   GPIO34   GPIO35    GPIO36
```

任何一个检测到悬空（HIGH），小车就后退 + 转向，防止掉桌子。

## 怎么接线

| 传感器引脚 | 接哪里 |
|-----------|--------|
| VCC | 3.3V 或 5V |
| GND | 共地 |
| DO | ESP32 GPIO（34 / 35 / 36） |

3 个传感器各占一个 GPIO，共用 VCC 和 GND。

## 代码

```cpp
#define CLIFF_L 34
#define CLIFF_M 35
#define CLIFF_R 36

bool cliffDetected() {
    // HIGH = 悬空 = 到桌边了
    return digitalRead(CLIFF_L) == HIGH
        || digitalRead(CLIFF_M) == HIGH
        || digitalRead(CLIFF_R) == HIGH;
}
```

## 调试技巧

- 模块上有个**小蓝色旋钮**（电位器），可以调灵敏度
- 把小车放在桌边，慢慢转旋钮，直到模块上的指示灯刚好在桌边亮/灭切换
- 深色桌面反射弱，可能需要调高灵敏度；白色桌面反射强，调低一点

## 一句话总结

**TCRT5000 就是小车的"悬崖眼睛"**，朝下发射红外线，检测到桌边（无反射）就输出 HIGH，小车收到信号立刻后退转向。3 个一组覆盖左中右，2–3 元一个。
