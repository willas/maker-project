# HC-SR04 超声波传感器

## 是什么

一个**测距模块**，朝前安装，用超声波测量前方障碍物的距离。像蝙蝠一样——发出超声波，听回声，算距离。

## 长什么样

```
大约 4.5cm × 2cm

    ┌───────────────────┐
    │                   │
    │   (○)      (○)    │ ← 左边发射超声波，右边接收回声
    │   Trig    Echo    │
    │                   │
    └───────────────────┘
     VCC  Trig  Echo  GND  ← 4 根线
```

两个"大眼睛"其实是超声波喇叭，一个发、一个收。

## 工作原理

```
    HC-SR04                    障碍物（水杯）
    (○)  ──→ 超声波 ──→  ──→    ┃
    (○)  ←── 回声   ←──  ←──    ┃
              │                  ┃
          测量时间差
     距离 = 时间 × 声速 / 2
```

ESP32 给 Trig 引脚一个 10 微秒的脉冲，模块发出超声波；回声到达时 Echo 引脚变高电平，高电平持续的时间 × 声速 ÷ 2 = 距离。

## 在清洁小车里怎么用

装在**车头正前方**，检测前方 2–400cm 范围内的障碍物（水杯、书本、笔筒等）。代码里设定：距离 < 8cm → 后退转向。

```
    小车前进方向 →

    [HC-SR04]  ─── 超声波 ───→  [水杯]
     "前方 6cm 有东西！"
     → 后退 + 转向
```

## 怎么接线

| 传感器引脚 | 接哪里 |
|-----------|--------|
| VCC | 5V |
| GND | 共地 |
| Trig | ESP32 GPIO 18（输出） |
| Echo | ESP32 GPIO 19（输入） |

## 代码

```cpp
#define TRIG_PIN 18
#define ECHO_PIN 19

float getDistanceCm() {
    // 发送 10 微秒脉冲
    digitalWrite(TRIG_PIN, LOW);
    delayMicroseconds(2);
    digitalWrite(TRIG_PIN, HIGH);
    delayMicroseconds(10);
    digitalWrite(TRIG_PIN, LOW);

    // 测量回声时间，换算成厘米
    long duration = pulseIn(ECHO_PIN, HIGH, 30000);
    if (duration == 0) return 999.0;  // 没收到回声 = 前方空旷
    return duration * 0.034 / 2.0;
}
```

## 注意事项

- 测量范围 2–400cm，但 2cm 以内测不准（太近了）
- 超声波有一定扩散角（约 15°），不是一根线，是个锥形区域
- 很薄的东西（如纸片）可能反射不回来，但水杯、书本完全没问题
- 这个传感器是**可选的**，就算不装，靠 3 个悬崖传感器也能防止掉桌子；加上它可以避免撞到桌上的东西
