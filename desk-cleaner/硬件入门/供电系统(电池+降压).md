# 供电系统（18650 电池 + LM2596 降压模块）

## 整体供电方案

```
2 节 18650 锂电池（串联 7.4V）
         │
         ├──→ TB6612 VM 引脚 → 驱动左右轮电机（直接用 7.4V）
         │
         └──→ LM2596 降压模块 → 输出 5V
                    │
                    ├──→ ESP32 VIN 引脚（主控供电）
                    ├──→ TB6612 VCC 引脚（驱动板逻辑供电）
                    ├──→ DFPlayer VCC（语音模块）
                    ├──→ MOS 管模块 V+（经 MOS 管给风扇、刷子电机）
                    └──→ 传感器 VCC（TCRT5000、HC-SR04 等）
```

## 18650 锂电池

### 是什么

一种**可充电锂电池**，直径 18mm、长度 65mm（所以叫 18650），笔记本电脑和充电宝里常用的就是它。

```
大约手指粗细，比 5 号电池大一圈

    ┌──┐
    │+ │ ← 正极（凸出来的那头）
    │  │
    │  │  单节电压 3.7V
    │  │  容量约 2000–3000mAh
    │  │
    │- │ ← 负极（平的那头）
    └──┘
```

### 为什么用 2 节串联

| 配置 | 电压 | 能不能用 |
|------|------|---------|
| 1 节 | 3.7V | 电压太低，电机没力气，降压后不够 5V |
| **2 节串联** | **7.4V** | 电机有力气，降压到 5V 给 ESP32 刚好 |
| 3 节串联 | 11.1V | 电压太高，N20 电机可能烧 |

### 选购要点

- 淘宝搜 `18650 锂电池 带保护板`，5–6 元/节
- **必须选带保护板的**（防过充、过放、短路）
- 买 2 节
- 另外买一个充电器：搜 `18650 充电器 双槽`，约 15 元

## 18650 电池盒

把 2 节电池装进去，串联输出 7.4V，带开关可以一键断电。

```
    ┌─────────────────────┐
    │ [电池1] [电池2]      │
    │  3.7V  + 3.7V       │
    │  = 串联 7.4V 输出    │
    │         [开关]       │
    └──┤红线(+) 黑线(-)├──┘
```

淘宝搜 `18650 双节电池盒 带开关`，2–3 元。

## LM2596 降压模块

### 是什么

把电池的 7.4V **降到稳定的 5V**，给 ESP32 和其他 5V 器件供电。电池电压会随放电慢慢降（7.4V → 6V），降压模块保证输出始终是 5V。

### 长什么样

```
大约 4cm × 2cm

    ┌──────────────────┐
    │  IN+ IN-         │ ← 输入（接电池 7.4V）
    │                  │
    │  [芯片] [电感]   │
    │        [旋钮] ←──│── 调节输出电压的小螺丝
    │                  │
    │  OUT+ OUT-       │ ← 输出（调到 5V）
    └──────────────────┘
```

### 怎么调到 5V

1. 输入端接电池（或任意 7–12V 电源）
2. 输出端接万用表
3. 用小螺丝刀转**旋钮**，看万用表读数，调到显示 **5.0V**
4. 调好后旋钮别再动，输出就锁定在 5V 了

没有万用表也行：先调到 ESP32 能正常启动（TYPE-C 口蓝灯亮），就差不多是 5V。

### 怎么接线

```
电池盒红线(+) ──→ LM2596 IN+
电池盒黑线(-) ──→ LM2596 IN-

LM2596 OUT+ ──→ ESP32 VIN + TB6612 VCC + DFPlayer VCC + MOS管 V+ + 传感器 VCC
LM2596 OUT- ──→ 所有 GND（共地）
```

## 一句话总结

**2 节 18650 串联出 7.4V 给电机，经 LM2596 降压到 5V 给 ESP32 和传感器。** 电池盒带开关可以一键断电，降压模块用旋钮调到 5V 就行。
