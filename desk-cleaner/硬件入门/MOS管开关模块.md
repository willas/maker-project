# MOS 管开关模块（D4184）

## 是什么

一个**电子开关**——ESP32 给一个小信号，MOS 管就帮你接通/断开大电流设备（风扇、刷子电机）。相当于用手指轻轻拨一下开关，就能控制一台大功率电器。

## 为什么需要它

ESP32 的 GPIO 引脚最大输出 40mA，而：
- 130 小电机（刷子）需要 200–500mA
- 3010 风扇需要 100–300mA

直接接会烧 ESP32。MOS 管模块做"中间人"：

```
ESP32 (小信号 3.3V)              大电流设备
   GPIO ──→ [MOS管模块] ──→ 130 电机 / 风扇
              ↑
           电池 (大电流)
```

## 长什么样

```
大约 2.5cm × 1.5cm

    ┌────────────────┐
    │   D4184 MOS    │
    │                │
    │ SIG  V+  V-   │ ← 输入端（接 ESP32 + 电源）
    │ OUT+ OUT-      │ ← 输出端（接电机/风扇）
    └────────────────┘
```

## 怎么接线

以刷子电机为例：

```
ESP32                MOS 管模块           130 电机
GPIO 12 ──────→ SIG（信号）
                V+ ←──────── 5V（电源正）
                V- ←──────── GND（共地）
                OUT+ ────────→ 电机红线(+)
                OUT- ────────→ 电机黑线(-)
```

风扇同理，用另一个 MOS 管模块（或共用一个，但分开控制更灵活）：

```
ESP32 GPIO 15 ──→ 第二个 MOS 管 ──→ 3010 风扇
```

## 代码

```cpp
#define BRUSH_PIN 12  // 刷子
#define FAN_PIN   15  // 风扇

// 刷子用 PWM 控制速度（可调快慢）
ledcWrite(CH_BR, 200);  // 速度 200/255
ledcWrite(CH_BR, 0);    // 关

// 风扇用 HIGH/LOW 控制开关（不需要调速）
digitalWrite(FAN_PIN, HIGH);  // 开
digitalWrite(FAN_PIN, LOW);   // 关
```

## 和继电器的区别

| | MOS 管模块 | 继电器 |
|--|---|---|
| 体积 | **很小** | 大 |
| 速度 | **极快**（可以 PWM 调速） | 慢（只能开/关，有机械延迟） |
| 噪音 | **无声** | 咔哒响 |
| 价格 | 2–3 元 | 3–5 元 |

MOS 管支持 PWM，所以能**调速**（比如调刷子转速），继电器只能开和关。清洁小车用 MOS 管更合适。

**MOS 管模块就是"ESP32 的力气放大开关"**，ESP32 给小信号，MOS 管接通电池大电流给电机/风扇。支持 PWM 调速，无声，2–3 元一个。
