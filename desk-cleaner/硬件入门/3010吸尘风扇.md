# 3010 吸尘风扇

## 是什么

一个**3cm × 3cm 的小风扇**，装在清洁小车里产生负压吸力，把毛刷扫起来的碎屑吸进收集盒。和电脑显卡上的小风扇类似，只是更小。

## 长什么样

```
3cm × 3cm × 1cm（"3010"就是 30mm × 10mm 的意思）

    ┌─────────────┐
    │  ╱ ╲   ╱ ╲  │
    │ │  ● ←轴  │ │ ← 扇叶
    │  ╲ ╱   ╲ ╱  │
    └─────────────┘
     红线(+)  黑线(-)
```

## 在清洁小车里怎么用

```
桌面碎屑 → [毛刷扫起] → [吸风口] → [风扇吸入] → [收集盒] → [滤网] → 空气排出
```

风扇装在收集盒和吸风口之间，产生负压（吸力），让碎屑跟着气流进入收集盒。滤网挡住碎屑，干净空气从背面排出。

## 怎么接线

和 130 电机一样，通过 **MOS 管模块**控制：

```
ESP32               MOS 管模块          3010 风扇
GPIO 15 ────────→  信号输入(SIG)
                    电源输入(V+) ←── 5V
                    GND ←───────── 共地
                    输出+ ─────────── 红线(+)
                    输出- ─────────── 黑线(-)
```

代码里：

```cpp
#define FAN_PIN 15

digitalWrite(FAN_PIN, HIGH);  // 风扇开
digitalWrite(FAN_PIN, LOW);   // 风扇关
```

## 选购

淘宝搜 `3010 直流风扇 5V` 或 `4010 直流风扇 5V`，3–5 元。
- **选 5V 的**（配降压模块输出的 5V）
- 3010 够用；如果想吸力更大可以选 4010（4cm × 4cm，稍大一号）
- 选带接线端子的（方便接线，不用焊）

## 一句话总结

**3010 风扇就是一个 3cm 的小吸尘器风扇**，产生负压把碎屑吸进收集盒，5V 供电，MOS 管控制开关，3–5 元。
