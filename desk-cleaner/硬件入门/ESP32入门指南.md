# ESP32 开发板入门指南

## ESP32 是什么

一块**指头大的微型电脑**，专门用来做智能硬件。可以理解为"比 Arduino 更强的 Arduino"——能写程序控制电机、读传感器、连 WiFi/蓝牙，只要 20 块钱。

## 长什么样

```
大约 5cm × 2.5cm，一张名片的 1/4 大小

        ┌──────────────────────┐
        │  ○ ○ ○ ○ ○ ○ ○ ○ ○ │ ← 左侧 GPIO 引脚（19个）
        │                      │
        │   ┌──────────┐       │
        │   │ ESP32    │       │ ← 金属屏蔽罩（芯片+天线在里面）
        │   │ WROOM-32 │       │
        │   └──────────┘       │
        │                      │
        │  ○ ○ ○ ○ ○ ○ ○ ○ ○ │ ← 右侧 GPIO 引脚（19个）
        │      [TYPE-C]        │ ← USB 口：供电 + 烧程序
        └──────────────────────┘
```

## 核心能力

| 能力 | 说明 | 在清洁小车里干什么 |
|------|------|-------------------|
| **38 个引脚 (GPIO)** | 可以接各种传感器、电机、LED | 接悬崖传感器、电机驱动板、蜂鸣器、DFPlayer 等 |
| **WiFi + 蓝牙** | 内置，不用额外模块 | 未来可以做手机遥控（本项目暂时不用） |
| **PWM 输出** | 可以控制电机速度（不只是开/关） | 调行走速度、刷子转速 |
| **ADC 模拟输入** | 读模拟传感器数值 | 暂时不用，预留扩展 |
| **5V 供电** | TYPE-C 口供电，或 VIN 引脚外接电源 | 用锂电池经降压模块供 5V |

## 和 Arduino 的对比

| | Arduino Nano | ESP32 |
|--|---|---|
| 价格 | ~15 元 | ~20 元 |
| 处理速度 | 16 MHz | **240 MHz**（快 15 倍） |
| WiFi/蓝牙 | 无 | **内置** |
| 引脚数 | 22 | **38** |
| 编程方式 | Arduino IDE | **也用 Arduino IDE**，代码几乎一样 |

ESP32 能做 Arduino 能做的所有事，还更快、引脚更多、自带 WiFi。价格只贵 5 块钱。

## 怎么用（3 步）

### 第 1 步：装软件

1. 电脑装 [Arduino IDE](https://www.arduino.cc/en/software)
2. 首选项 → 附加开发板管理器网址填：

```
https://dl.espressif.com/dl/package_esp32_index.json
```

3. 工具 → 开发板管理器 → 搜 `esp32` → 安装

### 第 2 步：接线

- TYPE-C 线连电脑（供电 + 传数据）
- 传感器/电机接到对应的 GPIO 引脚（就是板子两侧的针脚）
- 用杜邦线插就行，不用焊接

### 第 3 步：烧程序

1. Arduino IDE 里选开发板：`ESP32 Dev Module`
2. 选端口（插上线后会自动出现）
3. 打开 `desk_cleaner.ino`，点"上传"按钮
4. 等几秒，程序就烧进去了，板子自动开始运行

## 选购要点

淘宝搜 `ESP32-WROOM-32 开发板 TYPE-C`，注意选：

- **38 针**的（不是 30 针，引脚多更方便）
- **TYPE-C 口**的（比 Micro USB 耐用，正反都能插）
- 价格 15–25 元，不用买贵的

## 一句话总结

ESP32 就是一块 20 块钱的小电脑，用 Arduino IDE 写代码，TYPE-C 线连电脑烧程序，GPIO 引脚接传感器和电机，就能让清洁小车动起来。
