# TB6612FNG 电机驱动板

## 是什么

一块**电机驱动板**——ESP32 的引脚电流很小（最大 40mA），直接接电机根本带不动。TB6612 就是中间的"力气放大器"：ESP32 发指令，TB6612 把电池的大电流放出来驱动电机。

## 长什么样

```
大约 2cm × 2cm，比指甲盖大一点

    ┌─────────────────────┐
    │ VM  VCC GND STBY    │ ← 电源区
    │                     │
    │     TB6612FNG       │
    │                     │
    │ AIN1 AIN2 PWMA      │ ← A 通道（控制左电机）
    │ BIN1 BIN2 PWMB      │ ← B 通道（控制右电机）
    │ AO1  AO2  BO1  BO2  │ ← 电机输出
    └─────────────────────┘
```

## 为什么需要它

```
                    TB6612
ESP32 ──(小信号)──→ ┌─────┐ ──(大电流)──→ 电机转
                    │放大器│
电池 ──(电源)────→  └─────┘
```

ESP32 只负责说"左电机正转、速度 80%"，TB6612 负责把电池的电按这个要求送给电机。没有它，ESP32 直接接电机会烧板子。

## 引脚说明

| 引脚 | 接哪里 | 干什么 |
|------|--------|--------|
| **VM** | 电池 7.4V | 给电机供电（电机的力气从这来） |
| **VCC** | 降压后 5V | 给 TB6612 芯片自己供电 |
| **GND** | 共地 | ESP32、电池、TB6612 的 GND 接一起 |
| **STBY** | ESP32 GPIO 13 | 待机控制：HIGH = 工作，LOW = 休眠省电 |
| **AIN1** | ESP32 GPIO 25 | 左电机方向线 1 |
| **AIN2** | ESP32 GPIO 26 | 左电机方向线 2 |
| **PWMA** | ESP32 GPIO 32 | 左电机速度（PWM，0–255 控制快慢） |
| **BIN1** | ESP32 GPIO 27 | 右电机方向线 1 |
| **BIN2** | ESP32 GPIO 14 | 右电机方向线 2 |
| **PWMB** | ESP32 GPIO 33 | 右电机速度 |
| **AO1/AO2** | 左电机两根线 | 电机输出 A |
| **BO1/BO2** | 右电机两根线 | 电机输出 B |

## 怎么控制方向和速度

通过 IN1、IN2 两根线的高低电平组合控制方向：

| IN1 | IN2 | 电机状态 |
|-----|-----|---------|
| HIGH | LOW | **正转** |
| LOW | HIGH | **反转** |
| LOW | LOW | **停止** |

PWM 引脚控制速度：0 = 不动，255 = 全速。比如给 180 就是大约 70% 的速度。

对应到代码里就是这几行：

```cpp
// 前进：两个电机都正转
digitalWrite(AIN1, HIGH);
digitalWrite(AIN2, LOW);   // 左电机正转
digitalWrite(BIN1, HIGH);
digitalWrite(BIN2, LOW);   // 右电机正转
ledcWrite(PWMA, 180);      // 左电机速度 180/255
ledcWrite(PWMB, 180);      // 右电机速度 180/255

// 左转：左电机反转、右电机正转
digitalWrite(AIN1, LOW);
digitalWrite(AIN2, HIGH);  // 左电机反转
digitalWrite(BIN1, HIGH);
digitalWrite(BIN2, LOW);   // 右电机正转
```

## 和 L298N 的对比

| | TB6612FNG | L298N |
|--|---|---|
| 体积 | **2cm × 2cm**（很小） | 4cm × 4cm（大） |
| 效率 | **95%+**（MOSFET，几乎不发热） | ~50%（老芯片，发热大） |
| 电压损耗 | **0.5V** | 2–3V（电池电压浪费严重） |
| 驱动电流 | 1.2A/路（够 N20 电机） | 2A/路 |
| 价格 | 8–12 元 | 8–10 元 |

TB6612 体积小、效率高、不发热，对桌面清洁小车来说完全够用。

**TB6612 就是 ESP32 和电机之间的"翻译官"**——ESP32 用小信号说"往前走、速度 70%"，TB6612 把电池的大电流按这个要求送给电机。两个通道刚好驱动左右两个轮子。
