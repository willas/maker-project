# 130 小电机（清扫刷子用）

## 是什么

一个**最基础、最便宜的小马达**，用来带动清洁小车底部的**旋转毛刷**。和 N20 不同，130 电机没有减速箱，转速快、力气小，正好适合刷子高速旋转扫碎屑。

## 长什么样

```
大约 2.7cm × 2cm × 1.5cm，比大拇指粗一点

    ┌──────────────┐
    │              │
    │   130 马达   │──── 输出轴（细圆轴，直径 2mm）
    │              │
    └──┤+  -├──────┘
       两根线（红+、黑-）
```

小时候拆遥控车、电动玩具里面见到的那种小马达，就是 130 或它的近亲。

## 和 N20 的区别

| | 130 小电机 | N20 减速电机 |
|--|---|---|
| 有没有减速箱 | **没有**（裸电机） | 有（自带齿轮箱） |
| 转速 | **很快**（~8000–16000 RPM） | 慢（100–200 RPM） |
| 扭力 | 小 | **大** |
| 用途 | **带刷子**（要转得快才扫得动） | **驱动轮子**（要慢而有力） |
| 价格 | **1–2 元** | 6–10 元 |

**N20 驱动轮子（慢但有力），130 驱动刷子（快但不需要力气）**。

## 在清洁小车里怎么用

```
       130 电机
          │
          │ 输出轴
          ↓
    ┌───────────┐
    │  旋转毛刷  │ ← 刷毛朝下贴桌面
    └───────────┘
        ↓↓↓↓↓
      桌面碎屑被扫到吸风口
```

130 电机高速转 → 带动毛刷旋转 → 把橡皮屑从桌面扫起来 → 碎屑被后面的风扇吸进收集盒。

## 刷子怎么装到电机轴上

130 电机的轴是 **2mm 细圆轴**，毛刷需要固定上去：

| 方法 | 操作 | 推荐度 |
|------|------|--------|
| **热缩管** | 把刷轴套上去，用热风枪吹紧 | 最简单 |
| **3D 打印联轴器** | 打印一个中间件，一端卡电机轴，一端装刷毛 | 最牢固 |
| **瓶盖 DIY** | 小瓶盖钻 2mm 孔套在轴上，瓶盖外圈粘刷毛 | 最省钱 |
| **买现成刷头** | 淘宝搜 `2mm轴 小毛刷 清洁刷` | 最省事 |

## 怎么接线

130 电机**不接 TB6612**（TB6612 两个通道已经给了左右轮），而是通过一个 **MOS 管模块**由 ESP32 控制开关和速度：

```
ESP32               MOS 管模块          130 电机
GPIO 12 (PWM) ──→  信号输入(SIG)  ┐
                    电源输入(V+) ←── 电池 5V
                    GND ←───────── 共地
                    输出+ ─────────┤── 红线(+)
                    输出- ─────────┤── 黑线(-)
```

代码里就一行：

```cpp
#define BRUSH_PIN 12

// 刷子开（速度 200/255 ≈ 78%）
ledcWrite(CH_BR, 200);

// 刷子关
ledcWrite(CH_BR, 0);
```

PWM 值越大刷子转越快，可以调到合适的速度。

## 为什么不直接接 ESP32

130 电机工作电流 ~200–500mA，ESP32 引脚最大只能输出 40mA。MOS 管模块就是个**电子开关**，ESP32 给个小信号，MOS 管就把电池的大电流接通给电机。

```
ESP32: "刷子开，速度80%"（小信号）
  ↓
MOS管: "收到"（接通电池电流的 80%）
  ↓
130电机: 转起来（用电池的电，不是ESP32的电）
```

**130 小电机就是玩具车里那种便宜小马达**，转得快、没减速箱，用来高速旋转毛刷扫碎屑。1–2 块钱，通过 MOS 管模块接 ESP32 控制开关和速度。
