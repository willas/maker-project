# 语音台词录制指南

## 硬件

- **DFPlayer Mini** MP3 模块 + **Micro SD 卡** + **小喇叭**（8Ω 2W）— 语音播报（说话）
- **LU-ASR01** 语音识别模块 — 语音识别（听话）

> 两个模块分工：LU-ASR01 负责"听"（识别你说的指令），DFPlayer 负责"说"（播放可爱语音）。

## SD 卡准备

1. SD 卡格式化为 **FAT32**
2. 在 SD 卡根目录创建两个文件夹 `01` 和 `02`（两位数字命名）
3. 两个小朋友各录一套，分别放进自己的文件夹，**按编号命名**（必须 3 位数字）：

```
SD 卡/
├── 01/              ← 小朋友 A 的声音
│   ├── 001.mp3
│   ├── 002.mp3
│   ├── ...
│   └── 020.mp3
└── 02/              ← 小朋友 B 的声音
    ├── 001.mp3
    ├── 002.mp3
    ├── ...
    └── 020.mp3
```

> **注意文件名变化**：之前是 4 位数字（`0001.mp3`），现在用文件夹模式，改成 **3 位数字**（`001.mp3`）。这是 DFPlayer 的文件夹播放模式规定的。

## 切换声音

- **长按按钮 2 秒**：切换到另一个小朋友的声音（绿灯闪 = A，红灯闪 = B）
- **语音说"换声音"**：和长按效果一样（需在 LU-ASR01 中配置该关键词）
- 切换后会自动播放那个小朋友的开机语音，让你一听就知道是谁

## 录音方法

用手机录音 APP 录，然后转成 MP3（微信发给自己电脑也行）。
**两个小朋友各录一套完整的 20 条台词**，同样的内容用不同的声音录，比赛时可以切换展示。

> 💡 小贴士：两个人可以用不同的风格录——比如一个用萌萌的声音，一个用元气满满的声音，效果更好玩！

## LU-ASR01 语音识别 — 指令设置

LU-ASR01 负责"听"，识别到关键词后通过串口告诉 ESP32。

### 识别关键词列表

| 指令 ID | 关键词 | 触发动作 |
|---------|--------|---------|
| 0x01 | "打扫" / "开始" | 开始清扫（和按按钮效果一样） |
| 0x02 | "停下" / "停" | 停止清扫（和再按按钮效果一样） |
| 0x03 | "你好" | 回复开机语音（彩蛋） |
| 0x04 | "换声音" / "换一个" | 切换另一个小朋友的声音 |

### 怎么配置关键词

1. 电脑安装 **TIANWEN BLOCK**（天问 Block 图形化编程工具，模块店家提供下载链接）
2. Type-C 数据线连接 LU-ASR01 到电脑
3. 拖拽积木块设置：

```
当识别到 [打扫]  → 串口发送 [0x01]
当识别到 [开始]  → 串口发送 [0x01]
当识别到 [停下]  → 串口发送 [0x02]
当识别到 [停]    → 串口发送 [0x02]
当识别到 [你好]  → 串口发送 [0x03]
当识别到 [换声音] → 串口发送 [0x04]
当识别到 [换一个] → 串口发送 [0x04]
```

4. 点击一键烧录 → 完成（不需要写代码）

### LU-ASR01 接线

```
LU-ASR01 模块      ESP32
┌──────────┐
│ VCC      │ ←── 5V 或 3.3V
│ GND      │ ←── GND
│ TXD      │ ──→ GPIO 21（ESP32 RX1）
│ RXD      │ ←── GPIO 22（ESP32 TX1）
└──────────┘
```

### 比赛演示话术

现场演示时可以这样展示语音功能：

1. 对小车说 **"你好"** → 小车回复"你好呀！我是桌面小管家~"（小朋友 A 的声音）
2. 对小车说 **"打扫"** → 小车开始清扫，播放开场语音，跑起来
3. 清扫过程中小车自己说话、避障、防坠落
4. 对小车说 **"停下"** → 小车停止，说"好的好的，我休息一下~"
5. 对小车说 **"换声音"** → 灯闪烁，切换到小朋友 B 的声音，再打扫就用 B 的声音了
6. 也可以 **长按按钮 2 秒** 切换声音

> 不用按按钮，全程语音控制，评委看到会很惊艳。
> 两个小朋友各录了一套声音，现场切换展示，更能说明是团队协作的作品！

## DFPlayer Mini 接线

```
DFPlayer Mini        ESP32
┌──────────┐
│ VCC      │ ←── 5V（降压模块输出）
│ GND      │ ←── GND
│ RX       │ ←── GPIO 16（经 1kΩ 电阻）
│ TX       │ ──→ GPIO 17
│ SPK_1    │ ──→ 喇叭 +
│ SPK_2    │ ──→ 喇叭 -
└──────────┘
```

> RX 接 1kΩ 电阻是为了把 ESP32 的 3.3V 逻辑电平限流到 DFPlayer 的 5V 输入，防烧模块。

## Arduino 库安装

- DFPlayer：Arduino IDE → 库管理器 → 搜索 `DFRobotDFPlayerMini` → 安装
- LU-ASR01：不需要额外库（通过 HardwareSerial 串口通信，代码里已内置）
- LU-ASR01 关键词配置：安装 TIANWEN BLOCK（店家提供），图形化拖拽设置
