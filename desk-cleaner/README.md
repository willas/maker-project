# 智能桌面清洁小车

做完作业桌面一堆橡皮屑、纸屑？对小车说"打扫"，它就自动在桌面上跑，用旋转毛刷 + 吸尘风扇把碎屑收进收集盒，遇到桌边自动掉头不会掉下去，还会用可爱的声音和你聊天。说"停下"就乖乖停。

## 结构示意图

详见 `结构示意图.png`（俯视图 + 侧面图，可直接打印做海报）。

### 轮子布局：2 驱动轮 + 1 万向轮

```
             前进方向 ↑

        ◎    ◎    ◎     ← 3 个悬崖传感器（朝下）
       ═══════════════   ← 旋转毛刷
       ┌───────────────┐
       │               │
  左轮 ○               ○ 右轮    ← 2 个 N20 减速电机（驱动）
       │   [收集盒]    │
       │   [风扇]      │
       │   [ESP32]     │
       │   [电池]      │
       └───────┬───────┘
               ◎                 ← 1 个万向球轮（被动支撑）
```

- **左轮 + 右轮**：各接一个 N20 减速电机，差速控制转向（左快右慢 = 右转）
- **万向轮**：不带电机，只是一个小金属球轮，起支撑作用，让车身稳定
- 3 个着地点构成三角形，**最稳定、最简单**的底盘布局

### 清扫气流路径

```
    桌面碎屑
       ↓
  [旋转毛刷] 扫起碎屑
       ↓
  [吸风口] ← 风扇负压吸入
       ↓
  [收集盒] 碎屑留在盒中
       ↓
  [滤网 → 出风口] 空气排出
```

## 工作流程

```
说"打扫" 或 按下按钮
   ↓
播放开场语音 + 绿灯亮
   ↓
刷子开始转 + 风扇开始吸
   ↓
小车直线前进 ──→ 碎屑被刷进收集盒
   ↓
悬崖传感器检测到桌边？──→ 后退 + 随机转向 + "好险好险~"
超声波检测到障碍物？  ──→ 后退 + 转向 + "我绕过去~"
都没有？            ──→ 继续前进 + 随机说话/哼歌
   ↓
到达设定时间（默认 2 分钟）或 说"停下" / 再按一次按钮
   ↓
停止，红灯亮，播放结束语音
   ↓
拉出收集盒倒掉碎屑
```

## 烧录代码

1. 电脑安装 [Arduino IDE](https://www.arduino.cc/en/software)
2. Arduino IDE 里添加 ESP32 开发板支持：
   - 首选项 → 附加开发板管理器网址填：`https://dl.espressif.com/dl/package_esp32_index.json`
   - 工具 → 开发板管理器 → 搜索 `esp32` → 安装
3. 选择开发板：`ESP32 Dev Module`
4. 打开 `desk_cleaner.ino`，点上传

---

## 调试建议

- **先不装刷子和风扇**，只接两个行走电机 + 悬崖传感器，测试小车能不能在桌上走、到边缘会不会掉头。
- 调 `MOTOR_SPEED`（速度）和 `TURN_TIME_MS`（转弯时长），让小车跑得平稳。
- 确认行走没问题后，再装清扫电机和风扇。
- 串口监视器（115200）会打印状态，方便调试。

## 比赛展示建议

1. **现场语音演示**：对小车说"你好"→ 小车回应 → 说"打扫"→ 小车开始跑 → 撒橡皮屑看它扫 → 说"停下"→ 小车停。全程不用碰按钮，评委会很惊艳。
2. **多种加工方式**：
   - 编程（Arduino C++）
   - 电子（传感器焊接、电机接线）
   - 3D 打印（外壳、刷轴、收集盒）
   - 激光切割（底板/顶盖）
   - 手工组装（热熔胶固定、线缆整理）
3. **多种智能技术融合**：语音识别（LD3320）+ 语音播报（DFPlayer）+ 超声波避障 + 红外防坠 + 自主巡航
4. **创客精神**：说清楚"做完作业桌面很脏 → 想到做个自动清洁的东西 → 自己设计、自己做"。
5. **可带一份海报**：画上结构图、工作流程、照片。
