## 原清单对比：这次补了什么

| 新增项 | 原来为什么漏了 |
|--------|--------------|
| MOS 管模块 ×2 | 采购指引里有，但主清单表格忘了列 |
| 220Ω 电阻 ×2 | LED 限流必须有，不加可能烧 LED 或 GPIO |
| 1000μF 电容 ×2 | **(专家补充)** 解决电机启动瞬间拉低电压导致 ESP32 无限重启的问题 |
| 面包板 | 调试阶段必备，原清单只有洞洞板（焊接用的） |
| SD 卡读卡器 | 要往 SD 卡里拷 MP3 台词文件 |
| 18650 充电器 | 电池没电了需要充，之前完全漏了 |
| 公对公杜邦线 | 面包板两排之间连接需要，原来只有公对母和母对母 |
| 热熔胶枪 | 工具清单提到了但没放进采购清单 |
| 电烙铁 | 洞洞板焊接阶段必须用，之前完全没提 |
| 万用表 | 调降压模块电压必须有 |
| 排针（2.54mm 单排） | 最终组装洞洞板时需要，做接口排针用。淘宝 ¥1/条（40针） |
| 洞洞板（7×9cm） | 最终组装用，把 ESP32 和接线固定住。淘宝 ¥2-3/块 |
