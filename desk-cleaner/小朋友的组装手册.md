# 💡 第 1 关：点亮小灯！

> 🎯 任务：让 LED 灯亮起来，让蜂鸣器响起来，用按钮控制它们！
>
> 这是你第一次让电子零件"听你的话"——超级酷！

## 📦 你需要

```
┌────────────────────────────────────────────┐
│  从零件堆里拿出来：                          │
│                                            │
│  🧠 ESP32 开发板      ×1                   │

│                                            │
│  还需要：                                   │
│  💻 电脑（已装好 Arduino IDE）               │
└────────────────────────────────────────────┘
```

## 📋 步骤

### 第 ① 步：认识 ESP32 的"小脚"

ESP32 两边有很多小脚（引脚），每只脚都有一个编号。我们要用到的小脚：

```
                ESP32 开发板
    ┌─────────────────────────────────┐
    │ o                           o   │
    │ o  GPIO 2 → 接绿灯 💚       o   │
    │ o  GPIO 4 → 接红灯 ❤️       o   │
    │ o  GPIO 5 → 接按钮 🔘       o   │
    │ o                           o   │
    │ o                           o   │
    │ o                           o   │
    │ o                           o   │
    │ o                           o   │
    │ o  GPIO 23 → 接蜂鸣器 🔔    o   │
    │ o                           o   │
    │ o                           o   │
    │ o  GND → 接所有零件的负极    o   │
    │ o                           o   │
    │         [ USB-C 口 ]            │
    └─────────────────────────────────┘
```

> 💡 **GND** 就是"地"，所有零件都要接一根线到 GND，就像所有电器都要接地线。

### 第 ② 步：接线！像拼积木一样

用杜邦线把零件和 ESP32 连起来。一根线一根线来，别急：

**🟢 先接绿色 LED：**

```
    ESP32                     绿色 LED
   ┌─────┐                   
   │GPIO2│──── 杜邦线 ────── 长脚（+）
   │     │                   短脚（-）──── 杜邦线 ──── GND
   └─────┘

   💡 小提示：LED 有一长一短两只脚
      长脚 = 正极（+）= 接 GPIO
      短脚 = 负极（-）= 接 GND
```

**🔴 再接红色 LED：**（一模一样，只是换成 GPIO 4）

```
    ESP32                     红色 LED
   ┌─────┐                   
   │GPIO4│──── 杜邦线 ────── 长脚（+）
   │     │                   短脚（-）──── 杜邦线 ──── GND
   └─────┘
```

**🔔 接蜂鸣器：**

```
    ESP32                     蜂鸣器
   ┌──────┐                   ┌────┐
   │GPIO23│──── 杜邦线 ────── │ +  │
   │  GND │──── 杜邦线 ────── │ -  │
   └──────┘                   └────┘

   💡 蜂鸣器上面印着 + 和 - ，别接反了！
```

**🔘 接按钮：**

```
    ESP32                     按钮
   ┌──────┐                  ┌─────┐
   │GPIO 5│──── 杜邦线 ────── │一只脚│
   │  GND │──── 杜邦线 ────── │另一脚│
   └──────┘                  └─────┘

   💡 按钮只有两只脚需要接线
      一脚接 GPIO 5，一脚接 GND
```

### 第 ③ 步：检查接线

对照下面的表格，一根线一根线检查：

```
  ✅ 检查清单：
  [ ] 绿色 LED 长脚 → GPIO 2
  [ ] 绿色 LED 短脚 → GND
  [ ] 红色 LED 长脚 → GPIO 4
  [ ] 红色 LED 短脚 → GND
  [ ] 蜂鸣器 (+)   → GPIO 23
  [ ] 蜂鸣器 (-)   → GND
  [ ] 按钮 一脚     → GPIO 5
  [ ] 按钮 另一脚   → GND
```

### 第 ④ 步：插上 USB 线，连电脑

```
    ESP32 ════ USB-C 线 ════ 💻 电脑

    ESP32 上面会亮一个小红灯 = 有电了！
```

### 第 ⑤ 步：烧录程序！

让大人帮你打开 Arduino IDE，选择：
- 开发板：`ESP32 Dev Module`
- 端口：选带 `usbserial` 或 `CP210x` 字样的

然后打开文件：`tests/test_1_led_button/test_1_led_button.ino`

点 **上传按钮** ▶️（右箭头图标）

```
    等待中...
    ████████████████████░░░░  上传中...
    ████████████████████████  上传成功！✅
```

### 第 ⑥ 步：玩起来！

上传成功后，你会看到：

```
    1. 开机灯光秀！💚❤️💚❤️💚❤️ 闪三下
    2. 然后红灯亮着 = 待机

    按一下按钮 🔘 → 💚 绿灯亮！"嘀~"（高音）
    再按一下   🔘 → ❤️ 红灯亮！"嘟~"（低音）
    再按一下   🔘 → 💚 绿灯亮！"嘀~"
    ……交替切换！
```

### 🎮 挑战任务（选做，做了额外加 ⭐）

打开代码，试试改这些数字，看看会怎样：

| 找到这行代码 | 改改看 | 会怎样？ |
|-------------|--------|---------|
| `tone(BUZZER_PIN, 1500, 200)` | 把 `1500` 改成 `3000` | 声音变___了！ |

| `delay(300)` | 把 `300` 改成 `100` | 灯闪得___了！ |

- [ ] 开机灯光秀正常闪

# 👁 第 3 关：装眼睛 — 再也不掉桌子！

> 🎯 任务：装 3 个"悬崖眼睛"，让小车看到桌子边缘就后退！
>
> 这是最重要的安全功能！装完这个，小车就能安全地在桌上跑了。

## 📦 你需要

```
┌────────────────────────────────────────────┐
│  从零件堆里拿出来：                          │
│                                            │
│  👁 TCRT5000 传感器 ×3                      │
│  🔌 杜邦线                                  │
│  🔫 热熔胶枪（大人来用）                     │
└────────────────────────────────────────────┘
```

## 📋 步骤

### 第 ① 步：理解原理

```
  传感器朝下看，发出红外线（看不见的光）：

  【安全！有桌面】              【危险！要掉了！】

     传感器                       传感器
     ↓↓↓ 发出红外线               ↓↓↓ 发出红外线
     ↑↑↑ 反射回来！               ......  射出去，没回来
   ━━━━━━━━ 桌面                       ┃ 桌子边
                                       ┃
                                    (空气不反射)

     → 继续走！                    → 后退！转弯！
```

### 第 ② 步：👨 大人安装传感器

用热熔胶把 3 个传感器粘在小车前方底部，朝下：

```
              前进方向 ↑

         👁─────👁─────👁    ← 3 个传感器，朝下
         左     中     右       离桌面约 5-10mm
     ┌─────────────────────┐
     │                     │
     │       小车底部       │
```
