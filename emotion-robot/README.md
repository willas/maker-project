# 情绪安慰小机器人 · 树莓派版

用**摄像头**识别人脸表情，用**喇叭**播放安慰语音，适合科创比赛创客作品。

## 硬件

- **树莓派**（建议 4B 或 5，内存 2G+）
- **摄像头**（树莓派官方摄像头或 USB 摄像头）
- **喇叭**（3.5mm 耳机孔外接，或 USB 声卡 + 音箱）

## 安装（在树莓派上）

### 1. 系统

- 建议用 **64 位** Raspberry Pi OS（情绪识别库在 64 位下更稳定）。
- 开启摄像头：`sudo raspi-config` → Interface Options → Camera → Enable。
- 若有 3.5mm 喇叭，插上即可；USB 声卡需在系统里选为默认声卡。

### 2. 依赖

```bash
cd emotion-robot
python3 -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 3. 预录音（可选，效果更好）

在 `sounds/` 下放你自己录的安慰语，文件名对应情绪：

- `sad.wav` — 检测到「难过」时播放  
- `angry.wav` — 「生气」  
- `fear.wav` — 「害怕」  
- `neutral.wav` — 「平静」时打个招呼  

没有 WAV 时程序会用系统 **TTS（语音合成）** 读文字。

## 运行

```bash
source venv/bin/activate
python emotion_robot.py
```

- 把脸对准摄像头，程序会识别人脸和表情。
- 识别到需要安慰的情绪（如难过、生气）时，会播放对应安慰语（有间隔，不会一直说）。
- 按 **Q** 退出。

## 比赛展示建议

1. **多种加工方式**：外壳用 3D 打印或纸壳/亚克力，把树莓派、摄像头、喇叭装进去，做成「小机器人」造型。  
2. **创客精神**：自己录安慰语、自己设计外壳和表情规则。  
3. **演示**：先展示摄像头画面和识别出的情绪，再演示「难过时机器人说话安慰」。

## 可选购的「可编程小机器人」当身体

想让机器人不仅会说话、还会动（检测到难过时靠近、点头、转圈等），可以买**树莓派兼容的机器人底盘/小车套件**，把树莓派 + 摄像头 + 喇叭装上去，用 Python 控制电机。

| 类型 | 淘宝搜索关键词 | 参考价 | 说明 |
|------|----------------|--------|------|
| 两轮/四轮小车底盘 | `树莓派 智能小车 套件` `树莓派 机器人底盘` | 200–500 元 | 只有底盘+电机+驱动，需自备树莓派、摄像头、喇叭；GPIO 或 USB 控制，Python 好写 |
| 树莓派视觉小车（一体） | `树莓派 视觉小车 AI` `树莓派5 智能车` | 800–1500 元 | 带树莓派位、摄像头支架，装喇叭即可跑情绪识别，再加「检测到难过就前进」等动作逻辑 |
| 麦克纳姆轮小车 | `树莓派 麦克纳姆轮` | 400–800 元 | 可横移、转圈，动作更丰富；品牌如 Freenove、亚博智能等有教程和示例代码 |

**接法**：在 `emotion_robot.py` 里检测到 `sad` / `angry` 等时，除了播放语音，再调用电机控制（如 GPIO 或 驱动板串口），让小车前进一小段或转一圈，效果会更好。底盘商家一般会提供 Python 示例（前进、后退、左转、右转），直接复用即可。

---

## 若树莓派太卡

- 在 `emotion_robot.py` 里把 `CAMERA_WIDTH/CAMERA_HEIGHT` 改小（如 320×240）。  
- 或把 `CHECK_INTERVAL_SEC` 改大（如 5 秒检测一次）。  
- 也可以先在**电脑 + 摄像头**上跑通，再搬到树莓派。
