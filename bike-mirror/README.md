# 自行车智能盲区预警后视镜

> 小学生科创比赛项目 — 左右车把各装一个可折叠传感器臂，语音控制展开/收起，超声波检测后方来车，LED 提醒骑车人。

## 项目简介

骑自行车时回头看很危险，尤其是小朋友。这个装置模仿**汽车电动折叠后视镜**：

- 左右车把各装一个传感器臂，朝后检测盲区
- 说"打开"→ 两个臂自动旋转展开；说"收起"→ 自动折叠贴紧车把
- 检测到后方有车/人靠近 → LED 变色 + 蜂鸣器提醒
- 不骑车时折叠收起，不碍事

### 预警逻辑

| 状态 | 后方距离 | LED 表现 | 蜂鸣器 |
|------|---------|---------|--------|
| 安全 | > 4m 或无物体 | 绿灯常亮 | 静音 |
| 注意 | 2–4m 且在靠近 | 黄灯慢闪 | 静音 |
| 危险 | < 2m 且在靠近 | 红灯快闪 + 对应侧闪烁 | 嘀嘀响 |

左右独立检测——左边来车左边闪，右边来车右边闪。

## 折叠机制

```
【收起状态】臂贴着车把                【展开状态】臂向外旋转 90°

        车把                                   车把
  ──────┬──────                          ──────┬──────
  ▮臂▮  │  ▮臂▮  ← 贴紧                  臂   │   臂
        │                              ↙  (○) │ (○)  ↘
        │                          传感器朝后   │   传感器朝后
```

用 SG90 舵机驱动，0° = 收起，90° = 展开。和汽车电动折叠后视镜一个原理。

## 语音控制

使用 **LD3320 语音识别模块**（离线识别，不需要联网）：

| 语音指令 | 动作 |
|---------|------|
| "打开" / "展开" | 两侧臂旋转到 90°，开始检测 |
| "关闭" / "收起" | 两侧臂旋转到 0°，停止检测，绿灯常亮 |

备用：车把上有一个按钮，按一下也能切换展开/收起。

## 烧录步骤

1. Arduino IDE → 开发板选 `ESP32 Dev Module`
2. 库管理器安装 `ESP32Servo`
3. 打开 `bike_mirror.ino`
4. USB 连接 ESP32 → 选对端口 → 上传
