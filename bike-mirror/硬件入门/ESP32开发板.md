# ESP32 开发板（后视镜项目用法）

> 完整入门指南见 `../../desk-cleaner/硬件入门/ESP32入门指南.md`，本文只说后视镜项目的差异。

## 是什么

整个后视镜的"大脑"，负责：
- 读取左右两个 HC-SR04 的距离数据
- 接收 LD3320 语音识别结果
- 控制两个 SG90 舵机展开/收起
- 驱动 6 个 LED + 蜂鸣器

## 在后视镜里的 GPIO 用量

```
已用 GPIO：
  传感器:  18, 19, 32, 33  (左右 HC-SR04 各 2 个)
  舵机:    25, 26          (左右 SG90)
  LED:     12, 14, 27, 15, 2, 4  (左右各绿黄红)
  蜂鸣器:  23
  按钮:    5
  语音:    16, 17          (LD3320 串口)

  共用 16 个 GPIO，ESP32 的 38 针完全够用。
```

## 和清洁小车项目的区别

| | 后视镜项目 | 清洁小车项目 |
|--|---|---|
| 用了多少 GPIO | 16 个 | 20 个 |
| 串口 | Serial2 → LD3320 语音 | Serial2 → DFPlayer 语音 |
| PWM | 2 路（舵机） | 3 路（电机驱动） |
| 额外库 | `ESP32Servo` | `DFRobotDFPlayerMini` |

ESP32 都一样，只是接的东西不同。如果已经给清洁小车买了一块，可以先共用调试。

## Arduino IDE 设置

和清洁小车完全一样：
1. 开发板管理器安装 ESP32
2. 选 `ESP32 Dev Module`
3. 库管理器安装 `ESP32Servo`
4. 选对 COM 端口 → 上传
