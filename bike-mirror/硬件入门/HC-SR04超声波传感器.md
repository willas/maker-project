# HC-SR04 超声波传感器（后视镜项目用法）

> 完整入门指南见 `../../desk-cleaner/硬件入门/HC-SR04超声波传感器.md`，本文只说后视镜项目的差异。

## 是什么

用超声波测量后方物体距离的"眼睛"。和清洁小车里检测前方障碍物一样的模块，在后视镜项目里**朝后安装，检测后方来车/来人**。

## 和清洁小车的区别

| | 后视镜项目 | 清洁小车项目 |
|--|---|---|
| 数量 | **2 个**（左右各一） | 1 个（可选） |
| 安装方向 | **朝后**（检测来车） | 朝前（检测障碍物） |
| 安装位置 | 传感器臂前端 | 车头正前方 |
| 用途 | 盲区来车检测 | 避障 |

## 接线（2 个传感器）

```
左 HC-SR04:                     右 HC-SR04:
  VCC  → 5V                      VCC  → 5V
  GND  → GND                     GND  → GND
  Trig → GPIO 18                 Trig → GPIO 32
  Echo → GPIO 19                 Echo → GPIO 33
```

两个传感器共用 5V 和 GND，各自独立占 2 个 GPIO。

## 代码差异

清洁小车只测一次，后视镜需要**多次采样取中位数**（骑车颠簸会产生噪音），并判断**距离是否在缩小**（靠近检测）：

```cpp
float dist = measureDist(L_TRIG, L_ECHO);  // 3 次采样取中位数
float delta = dist - prevDist;               // 距离变化
bool approaching = (delta < -5);             // 缩小 > 5cm = 在靠近
```

## 安装方式

HC-SR04 装在 3D 打印传感器臂的**前端**，两个"眼睛"朝后：

```
    臂根部 ────────── 臂前端
    (舵机)              (○)(○) ← HC-SR04 朝后
                           ↓
                       后方来车方向
```

用 M2 螺丝或热熔胶固定。传感器的 4 根线沿着臂走线到车把主控盒。
