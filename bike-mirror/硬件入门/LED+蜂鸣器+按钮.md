# LED + 蜂鸣器 + 按钮（后视镜项目用法）

> 完整入门指南见 `../../desk-cleaner/硬件入门/LED+蜂鸣器+按钮.md`，本文只说后视镜项目的差异。

## LED 指示灯

### 和清洁小车的区别

| | 后视镜项目 | 清洁小车 |
|--|---|---|
| 数量 | **6 个**（左右各绿黄红） | 2 个（红绿） |
| 作用 | 三级预警（安全/注意/危险） | 状态指示（清扫/停） |
| 安装位置 | 传感器臂顶部（骑车时可见） | 车身顶部 |

### 接线

```
左侧 LED:                          右侧 LED:
  GPIO 12 → [220Ω] → 绿 LED → GND   GPIO 15 → [220Ω] → 绿 LED → GND
  GPIO 14 → [220Ω] → 黄 LED → GND   GPIO  2 → [220Ω] → 黄 LED → GND
  GPIO 27 → [220Ω] → 红 LED → GND   GPIO  4 → [220Ω] → 红 LED → GND
```

每个 LED 串一个 220Ω 电阻限流，和清洁小车完全一样。

### 预警逻辑

```
后方 > 4m        → 绿灯常亮         （安全）
后方 2-4m 且靠近  → 黄灯慢闪(500ms)  （注意）
后方 < 2m 且靠近  → 红灯快闪(120ms)  （危险）
```

左右独立——左边来车只左边亮，右边来车只右边亮。

---

## 有源蜂鸣器

和清洁小车完全一样：

```
ESP32 GPIO 23 → 蜂鸣器 SIG
                蜂鸣器 GND → GND
```

在后视镜项目中，只有**两侧任一出现"危险"（红灯）时蜂鸣器才响**，跟着红灯快闪节奏"嘀嘀嘀"。

---

## 按钮

和清洁小车完全一样：

```
ESP32 GPIO 5 (内部上拉) → 按钮 → GND
```

在后视镜项目中，按钮是**语音控制的备用方案**：
- 按一下 → 臂展开（等同于说"打开"）
- 再按一下 → 臂收起（等同于说"收起"）

装在车把主控盒上，骑车时大拇指能按到的位置。

---

## 选购

- LED：淘宝搜 `5mm LED 高亮 红绿黄 散装`，买绿×2 黄×2 红×2
- 220Ω 电阻：`220欧 电阻 直插`，买 6 个
- 有源蜂鸣器：`有源蜂鸣器模块 Arduino`，2 元
- 按钮：`轻触开关 6x6` 或 `按钮模块 Arduino`
