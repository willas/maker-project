# SG90 舵机（9g 微型舵机）

## 是什么

一个**能精确转到指定角度的小马达**。普通电机只会"转转转"停不下来，舵机可以说"转到 90°"它就精确转到 90° 停住。遥控车转向、机械臂关节、航模舵面用的都是它。

在这个项目里，SG90 驱动传感器臂**折叠/展开**——0° 收起贴着车把，90° 展开朝后检测。

## 长什么样

```
大约 2.3cm × 1.2cm × 2.5cm，比大拇指还小，重量只有 9 克

    ┌─────────┐
    │  SG90   │
    │         │──── 白色舵盘（输出轴，转动的部分）
    │         │
    └────┬────┘
         │
    三根线：棕(GND) 红(VCC) 橙(信号)
```

### 三根线

| 线色 | 作用 | 接哪里 |
|------|------|--------|
| **棕色** | GND | ESP32 GND |
| **红色** | VCC 供电 | 5V（降压模块输出） |
| **橙色** | 信号线 | ESP32 GPIO（25 或 26） |

## 工作原理

ESP32 通过 PWM 信号告诉舵机要转到多少度：

```
ESP32 发送 PWM 脉冲：
  脉宽 0.5ms → 舵机转到 0°  （收起）
  脉宽 1.5ms → 舵机转到 90° （中间）
  脉宽 2.5ms → 舵机转到 180°

本项目只用 0° 和 90°：
  0°  = 臂贴着车把（收起）
  90° = 臂向外展开（检测状态）
```

不需要自己算脉宽，Arduino 的 `Servo` 库帮你搞定：

```cpp
#include <ESP32Servo.h>

Servo myServo;
myServo.attach(25);     // 信号线接 GPIO 25
myServo.write(0);       // 转到 0°（收起）
myServo.write(90);      // 转到 90°（展开）
```

## 和普通电机的区别

| | SG90 舵机 | 普通电机（如 N20、130） |
|--|---|---|
| 能不能转到指定角度 | **能**（0°–180° 任意角度） | 不能（只会一直转） |
| 控制方式 | PWM 信号指定角度 | 电压控制转速 |
| 转动范围 | 0°–180°（半圈） | 360°（连续转） |
| 用途 | 转向、摆臂、机械臂关节 | 驱动轮子、风扇 |
| 自带减速齿轮 | **有**（内部齿轮箱） | N20 有，130 没有 |

## 在自行车后视镜里怎么用

```
【收起状态 0°】              【展开状态 90°】

  车把 ────┬────              车把 ────┬────
     ▮臂▮ │                       │
   SG90→  ●                  SG90→  ● ── 臂 ── (○○) HC-SR04
           │                       │        ↑ 朝后检测
```

每侧车把末端装一个 SG90：
- 语音说"打开"→ `servo.write(90)` → 臂旋转展开
- 语音说"收起"→ `servo.write(0)` → 臂折叠回来

### 平滑旋转

直接 `write(90)` 舵机会"嗖"一下转过去，不好看。代码里做了平滑处理：

```cpp
// 每 20ms 转 2°，总共 90° 需要约 0.9 秒
// 看起来像汽车电动折叠后视镜的速度
int currentAngle = 0;
int targetAngle = 90;

void loop() {
    if (currentAngle < targetAngle) {
        currentAngle += 2;
        servo.write(currentAngle);
        delay(20);
    }
}
```

## 安装方式

```
    车把末端（圆管）
         │
    ┌────┴────┐
    │ 3D打印  │ ← 夹座（夹在车把管上）
    │  夹座   │
    │  ┌───┐  │
    │  │SG90│  │ ← 舵机用 M2 螺丝固定在夹座里
    │  └─┬─┘  │
    └────┼────┘
         │
    ┌────┴────────────┐
    │   3D打印传感器臂  │ ← 臂根部套在舵盘上
    │              (○○)│ ← HC-SR04 装在臂前端
    └─────────────────┘
```

舵盘（白色十字/圆盘）用螺丝固定到臂的根部。舵机转 → 舵盘转 → 臂跟着转。

## 注意事项

- **不要用手硬掰**：舵机通电时有保持力，硬掰会烧齿轮
- **供电要够**：SG90 运动时瞬间电流可达 500mA，用降压模块的 5V（不要直接用 ESP32 的 3.3V）
- **选金属齿轮版**：塑料齿轮版便宜但容易打齿，金属齿轮版多 1-2 元但耐用得多
- **舵盘别装反**：先 `write(0)` 让舵机归零，再把舵盘装到臂收起的位置
