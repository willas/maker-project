# 供电系统（后视镜项目用法）

> 完整入门指南见 `../../desk-cleaner/硬件入门/供电系统(电池+降压).md`，本文只说后视镜项目的差异。

## 方案概览

和清洁小车**完全一样**的供电方案：2 节 18650 串联 → LM2596 降压到 5V。

```
2 节 18650（串联 7.4V）
         │
         └──→ LM2596 降压 → 5V
                    │
                    ├── ESP32 VIN
                    ├── 左右 HC-SR04 VCC
                    ├── 左右 SG90 舵机 VCC（红线）
                    ├── LD3320 VCC
                    └── 蜂鸣器 VCC
```

## 和清洁小车的区别

| | 后视镜项目 | 清洁小车 |
|--|---|---|
| 电池数 | 2 节（一样） | 2 节 |
| 总功耗 | **低**（无电机，峰值 ~1A） | 高（电机+风扇，峰值 ~3A） |
| 续航 | **10+ 小时** | 1–2 小时 |
| 体积要求 | **更小**（要装在车把上） | 宽松（装在车体内） |

后视镜项目最耗电的是两个 SG90 舵机（展开/收起时各 ~500mA），但它们只在动的那 1 秒钟耗电，平时几乎不耗电。所以整体续航非常长。

## 为什么不用 1 节电池 + 升压

单节 18650（3.7V）+ MT3608 升压到 5V 理论上可行，但：
- SG90 舵机堵转瞬间电流大，升压模块可能电流不够
- 2 节串联降压更稳定、更安全

如果特别想要更小体积（只装 1 节电池），可以用 MT3608 升压模块 + 大容量 18650（3500mAh），但需要实测舵机能不能正常工作。

## 安装位置

电池盒装在车把中央的主控盒里，和 ESP32 放在一起：

```
    ┌── 车把主控盒 ──┐
    │  [ESP32]       │
    │  [LD3320]      │
    │  [洞洞板]       │
    │  [电池盒+2节]   │
    │  [LM2596]      │
    │  [开关]         │
    └────────────────┘
```

## 选购

和清洁小车完全一样：
- 18650 锂电池：`18650 锂电池 带保护板`，5–6 元/节 × 2
- 电池盒：`18650 双节电池盒 带开关`，2–3 元
- LM2596 降压：`LM2596 DC-DC 降压模块 可调`，2–3 元

> 如果已经给清洁小车买了这些，可以直接复用调试，比赛时再各买一套。
